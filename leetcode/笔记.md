二分法大致模板: 
```java

        while (left <= right) {
            // int mid = (left + right) / 2;
            int mid = left + ((right - left) >> 1); // 防止溢出
            int guess = guess(mid);
            if (guess == 0) return mid;
            else if (guess == -1) { // 大了
                right = mid - 1;
            } else if (guess == 1) {
                left = mid + 1;
            }
        }
        return right - left + 1;
```

最大公约数模板: 36 , 48
```java
    private int gcd(int a, int b) {
        if (b == 0) {
            return a;
        }
        return gcd(b, a % b);
    }
```
